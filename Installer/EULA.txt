2. PRIVACY NOTICE
-----------------
BKKleaner respects your privacy:

  - BKKleaner does NOT collect, transmit or sell any personal data.
  - All settings, logs, backups and benchmark reports are stored locally in
    %LOCALAPPDATA%\BKKleaner on your computer.
  - The only network requests BKKleaner makes are:
      * an optional check to GitHub for a newer application version;
      * optional package update queries through the Windows Package Manager
        (winget) when you explicitly use the Updates page.
  - BKKleaner requires administrator rights to read hardware sensors and to
    apply safe, reversible Windows optimizations. Every optimization is backed
    up first (registry export, restore point, snapshot) and can be undone.

3. SAFE OPERATION
-----------------
BKKleaner only performs documented, reversible changes. It never deletes
system files, never force-closes protected processes, and quarantines cleaned
temporary files so they can be restored.
